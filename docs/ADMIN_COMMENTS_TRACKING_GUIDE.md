# دليل الأدمن: تتبع التعليقات المتكررة
## Admin Guide: Tracking Repeated Comments

---

## 🎯 الهدف

تتبع المناديب الذين يكررون نفس المشاكل (إجازات متكررة، حوادث، غيابات، إلخ) واتخاذ الإجراءات المناسبة.

---

## 📊 كيفية استخدام لوحة التعليقات

### **1. افتح لوحة التحكم**

```
URL: /admin/rider-comments-dashboard
```

### **2. فلترة التعليقات**

#### **حسب التاريخ**:
```
من تاريخ: 2026-07-01
إلى تاريخ: 2026-07-31
← عرض كل التعليقات في يوليو
```

#### **حسب الفئة**:
```
الفئة: إجازة مرضية 🏥
← عرض الإجازات المرضية فقط
```

#### **حسب المشرف**:
```
المشرف: محمد فؤاد
← عرض تعليقات مشرف معين
```

---

## 🚨 حالات التحذير

### **حالة 1: مندوب كثير الإجازات**

**المعايير**:
- 3+ إجازات مرضية في الشهر
- أو 7+ أيام إجازة في الشهر

**كيفية الكشف**:
```
1. افتح لوحة التحكم
2. اختر آخر 30 يوم
3. اختر الفئة: إجازة مرضية 🏥
4. انظر إلى جدول "تكرار التعليقات"
5. المناديب الذين لديهم 3+ تعليقات → تحذير!
```

**الإجراء**:
1. طلب شهادة طبية
2. فحص طبي شامل
3. إنهاء عقد إذا لزم الأمر

---

### **حالة 2: مندوب كثير الحوادث**

**المعايير**:
- 2+ حوادث في الشهر

**كيفية الكشف**:
```
1. افتح لوحة التحكم
2. اختر آخر 30 يوم
3. اختر الفئة: حادث 🚑
4. انظر إلى جدول "تكرار التعليقات"
5. المناديب الذين لديهم 2+ حوادث → تحذير!
```

**الإجراء**:
1. تدريب سلامة إلزامي
2. تقييم قيادة السكوتر
3. خصم تأمين إضافي
4. إنهاء عقد إذا تكرر

---

### **حالة 3: مندوب كثير الغيابات**

**المعايير**:
- 3+ غيابات متكررة في الأسبوع

**كيفية الكشف**:
```
1. افتح لوحة التحكم
2. اختر آخر 7 أيام
3. اختر الفئة: غيابات متكررة ⚠️
4. انظر إلى جدول "تكرار التعليقات"
5. المناديب الذين لديهم 3+ غيابات → تحذير!
```

**الإجراء**:
1. تحذير رسمي أول
2. تحذير رسمي ثاني
3. إنهاء عقد

---

### **حالة 4: مندوب أداء ضعيف متكرر**

**المعايير**:
- 5+ أيام أداء ضعيف في الشهر

**كيفية الكشف**:
```
1. افتح لوحة التحكم
2. اختر آخر 30 يوم
3. اختر الفئة: أداء ضعيف 📉
4. انظر إلى جدول "تكرار التعليقات"
5. المناديب الذين لديهم 5+ أيام → تحذير!
```

**الإجراء**:
1. تدريب أداء
2. خطة تحسين (30 يوم)
3. إنهاء عقد إذا لم يتحسن

## 📈 أمثلة واقعية

### **مثال 1: مندوب يأخذ إجازة كل أسبوع**

**البيانات**:
```
أحمد محمد (877614)
التعليقات في يوليو:
- 2026-07-01: إجازة مرضية 🏥 (يومان)
- 2026-07-08: إجازة مرضية 🏥 (3 أيام)
- 2026-07-15: إجازة مرضية 🏥 (يوم)
- 2026-07-22: إجازة مرضية 🏥 (يومان)
- 2026-07-29: إجازة مرضية 🏥 (يوم)

إجمالي: 5 إجازات، 9 أيام غياب
```

**التحليل**:
- معدل: إجازة كل أسبوع
- أيام الغياب: 30% من الشهر
- التقييم: 🔴 حرج

**القرار**:
1. مطالبة بشهادة طبية لكل إجازة
2. فحص طبي شامل
3. إذا لم توجد أسباب طبية → إنهاء عقد

---

### **مثال 2: مندوب كثير الحوادث**

**البيانات**:
```
علي حسن (123456)
التعليقات في 3 أشهر:
- 2026-05-10: حادث 🚑 - سقط من السكوتر
- 2026-06-15: حادث 🚑 - اصطدام بسيارة
- 2026-07-20: حادث 🚑 - انزلاق

إجمالي: 3 حوادث في 3 أشهر
```

**التحليل**:
- معدل: حادث كل شهر
- التكلفة: مرتفعة (إصلاحات + إجازات)
- التقييم: 🔴 حرج

**القرار**:
1. تدريب سلامة إلزامي (40 ساعة)
2. اختبار قيادة
3. خصم تأمين إضافي 20%
4. تحذير: حادث رابع = إنهاء عقد

## 🔍 كيفية البحث السريع

### **البحث عن مندوب معين**:

```
1. افتح لوحة التحكم
2. في جدول "تكرار التعليقات"
3. اضغط Ctrl + F
4. ابحث عن اسم المندوب
5. انظر إلى "إجمالي التعليقات" و "التفاصيل"
```

### **البحث عن مشرف معين**:

```
1. افتح لوحة التحكم
2. في فلتر "المشرف"
3. اختر المشرف
4. انظر إلى كل تعليقاته
5. هل يسجل التعليقات بانتظام؟ ← تقييم المشرف
```

---

## 📊 التقارير الشهرية

### **في نهاية كل شهر، راجع**:

#### **1. المناديب الأكثر تعليقات**:
```
الترتيب:
1. أحمد محمد: 12 تعليق
2. علي حسن: 8 تعليقات
3. محمد علي: 6 تعليقات

← هؤلاء يحتاجون متابعة خاصة
```

#### **2. الفئات الأكثر تكراراً**:
```
توزيع التعليقات:
- إجازة مرضية: 45 تعليق
- حادث: 23 تعليق
- أداء ضعيف: 18 تعليق
- غيابات متكررة: 15 تعليق

← إجراءات: تحسين الصحة، تدريب سلامة
```

#### **3. المشرفون الأكثر تسجيلاً**:
```
الترتيب:
1. محمد فؤاد: 50 تعليق ← ممتاز
2. أحمد علي: 35 تعليق ← جيد
3. علي محمد: 5 تعليقات ← ضعيف (لا يسجل بانتظام)

← تحذير المشرفين الذين لا يسجلون
```

## ⚡ نصائح سريعة

### **للكشف السريع عن المشاكل**:

```bash
# كل صباح، افتح لوحة التحكم وراجع:

1. التعليقات خلال آخر 7 أيام
   → هل هناك مناديب متكررين؟
   
2. توزيع الفئات
   → هل زادت الحوادث؟ الإجازات؟
   
3. المناديب المتوقع عودتهم اليوم
   → متابعة من المشرف
```

## 🎯 KPIs للتتبع

### **مؤشرات الأداء الرئيسية**:

```
1. معدل التعليقات شهرياً:
   - الهدف: < 2 تعليق/مندوب/شهر
   - الحرج: > 4 تعليقات/مندوب/شهر

2. معدل الحوادث:
   - الهدف: < 5 حوادث/100 مندوب/شهر
   - الحرج: > 10 حوادث/100 مندوب/شهر

3. معدل الإجازات المرضية:
   - الهدف: < 3 أيام/مندوب/شهر
   - الحرج: > 7 أيام/مندوب/شهر

4. معدل الغيابات:
   - الهدف: < 2 غياب/مندوب/شهر
   - الحرج: > 4 غيابات/مندوب/شهر
```

## 🚀 التكامل مع Strategic Ops (قريباً)

سيتم دمج التعليقات مع مركز العمليات الاستراتيجي لعرض:

- **المناديب الأكثر خطورة**: حوادث + إجازات + غيابات
- **التوصيات التلقائية**: بناءً على التكرار
- **تكلفة المشاكل**: حساب التكلفة الإجمالية للحوادث والإجازات
- **توقع المشاكل**: AI للتنبؤ بالمناديب الذين قد يواجهون مشاكل

## ✅ الخلاصة

**نظام التعليقات اليومية يوفر**:

✅ **تتبع دقيق** لكل تعليق حسب التاريخ
✅ **كشف تلقائي** للمناديب المتكررين في المشاكل
✅ **فلترة متقدمة** حسب التاريخ والفئة والمشرف
✅ **تحذيرات واضحة** عند التكرار
✅ **سجل دائم** لا يمكن حذفه
✅ **توصيات واضحة** للإجراءات المطلوبة
