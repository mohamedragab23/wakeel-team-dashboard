# 🎯 نظام تحليلات العمليات اللوجستية (Logistics Intelligence System)

## نظرة عامة

نظام متكامل لتحليل بيانات الأداء التشغيلي، تقييم المناديب والمشرفين، وتقديم توصيات ذكية لتحسين الكفاءة وتحقيق الأهداف الاستراتيجية.

---

## 📊 المميزات الرئيسية

### 1️⃣ **التحليلات الأساسية (Analytics)**

#### المؤشرات الرئيسية (KPIs)
- ✅ **إجمالي المناديب**: العدد الكلي للمناديب في الفترة المحددة
- ✅ **متوسط النشطين يومياً**: المناديب الذين عملوا فعلياً (orders > 0 & hours > 0)
- ✅ **متوسط الغياب يومياً**: المناديب الذين لم يعملوا (orders = 0 & hours = 0)
- ✅ **نسبة النشطين**: (النشطين / الإجمالي) × 100
- ✅ **متوسط ساعات العمل**: لكل مندوب في اليوم
- ✅ **متوسط ساعات الاستراحة**: لكل مندوب في اليوم
- ✅ **إجمالي ساعات العمل**: للفترة المحددة

#### شرائح الأداء (Performance Segments)
المناديب مقسمين حسب معدل ساعات العمل اليومية:

| الشريحة | التقييم | اللون |
|---------|---------|-------|
| أقل من 4 ساعات | 🔴 ضعيف | أحمر |
| 4-6 ساعات | 🟡 متوسط | برتقالي |
| 6-8 ساعات | 🔵 جيد | أزرق |
| 8+ ساعات | 🟢 ممتاز | أخضر |

**لكل شريحة:**
- عدد المناديب
- أسماء المناديب وأكوادهم
- متوسط ساعات العمل لكل مندوب

#### المناديب غير النشطين (Inactive Riders)
- كشف المناديب الذين لم يعملوا لمدة **3 أيام فأكثر**
- عرض عدد أيام الغياب لكل مندوب
- آخر تاريخ نشاط

#### أعلى المتخذين للاستراحات (Top Break Takers)
- ترتيب المناديب حسب متوسط ساعات الاستراحة اليومية
- عرض Top 10 مناديب
- إجمالي ساعات الاستراحة للفترة

#### أعلى الغائبين (Top Absent Riders)
- ترتيب المناديب حسب عدد أيام الغياب
- عرض Top 10 مناديب
- نسبة الغياب من الإجمالي

---

### 2️⃣ **حاسبة Target (Target Calculator)**

#### الهدف
حساب عدد المناديب النشطين المطلوبين لتحقيق Target يومي محدد من ساعات العمل.

#### المدخلات
- **Target اليومي**: مثال 1500 ساعة/يوم
- **متوسط ساعات العمل**: من التحليلات الحالية (مثال 5.9 ساعة/مندوب)

#### المخرجات
- **عدد المناديب المطلوب**: Target ÷ متوسط الساعات
- **الفجوة (Gap)**: المطلوب - الموجود حالياً
- **التوصيات**:
  - إذا كان Gap موجب: "تحتاج لتعيين X مندوب جديد"
  - إذا كان Gap سالب: "عندك فائض X مندوب"
  - توصيات فورية وطويلة المدى

#### مثال عملي
```
Target اليومي = 1500 ساعة
متوسط الساعات = 5.9 ساعة/مندوب
المناديب النشطين الحاليين = 200

عدد المناديب المطلوب = 1500 ÷ 5.9 = 254 مندوب
الفجوة = 254 - 200 = +54 مندوب

✅ التوصية: تحتاج لتعيين 54 مندوب جديد لتحقيق Target
```

---

### 3️⃣ **مولد خطط النمو (Growth Plan Generator)**

#### الهدف
إنشاء خطة استراتيجية مرحلية لتحقيق Target نهائي في إطار زمني محدد.

#### المدخلات
- **Target النهائي**: مثال 2000 ساعة/يوم
- **Target الحالي**: من البيانات الموجودة (مثال 1200 ساعة/يوم)
- **الإطار الزمني**: مثال 4 شهور

#### المخرجات
خطة مقسمة على مراحل شهرية تتضمن:

**لكل شهر:**
- **Target الساعات**: الهدف التدريجي
- **المناديب المطلوبين**: العدد الإجمالي
- **التعيينات الجديدة**: عدد المناديب المطلوب تعيينهم
- **هدف متوسط الساعات**: الهدف لكل مندوب
- **المعالم الرئيسية (Milestones)**:
  - أهداف التوظيف
  - أهداف الأداء
  - خطط التدريب

#### مثال عملي
```
Target النهائي = 2000 ساعة
Target الحالي = 1200 ساعة
الإطار الزمني = 4 شهور

الشهر 1:
  - Target: 1400 ساعة
  - المناديب: 237
  - تعيينات جديدة: +37
  - متوسط الساعات: 5.9
  - Milestones:
    ✓ توظيف 37 مندوب جديد
    ✓ تدريب المجموعة الجديدة
    ✓ تحسين متوسط الساعات بـ 3%

الشهر 2:
  - Target: 1600 ساعة
  - المناديب: 271
  - تعيينات جديدة: +34
  ...
```

### 4️⃣ **تحذيرات الأداء (Performance Warnings)**

#### الهدف
تحليل ذكي لتحديد المشاكل التشغيلية الحالية وتقديم حلول عملية.

#### أنواع التحذيرات

##### 🔴 **تحذيرات حرجة (Critical)**
مشاكل تتطلب تدخل فوري:
- نسبة غياب أعلى من 25%
- متوسط ساعات عمل أقل من 4.5 ساعة
- أكثر من 40% من المناديب يعملون أقل من 4 ساعات

##### 🟠 **تحذيرات مرتفعة (High)**
مشاكل تحتاج انتباه عاجل:
- نسبة غياب 15-25%
- متوسط ساعات عمل 4.5-5.5 ساعة
- أكثر من 30% من المناديب يعملون أقل من 4 ساعات

##### 🟡 **تحذيرات متوسطة (Medium)**
مشاكل تحتاج مراقبة:
- متوسط ساعات استراحة أعلى من 1.5 ساعة
- نسبة غياب 10-15%

#### لكل تحذير:
- **الفئة**: نوع المشكلة
- **الوصف**: تفاصيل المشكلة
- **التأثير**: الأثر على العمليات
- **الحلول المقترحة**: خطوات عملية للحل
- **الإطار الزمني**: متى يجب التنفيذ

#### مثال
```
🔴 تحذير حرج - نسبة الغياب العالية
المشكلة: نسبة الغياب 28% (56 مندوب من 200)
التأثير: خسارة 336 ساعة عمل يومياً (56 × 6 ساعات)

الحلول المقترحة:
  1. تحليل أسباب الغياب (اتصل بالمشرفين)
  2. تحسين الحوافز للحضور المنتظم
  3. مراجعة جداول الشفتات
  4. تعيين مناديب احتياطيين
  5. تطبيق نظام حضور صارم

الإطار الزمني: فوري (خلال 3 أيام)
```

### 5️⃣ **تقييم المشرفين (Supervisor Evaluation)**

#### المؤشرات المستخدمة
لكل مشرف يتم حساب:

1. **عدد المناديب**: إجمالي المناديب تحت إشرافه
2. **متوسط النشطين يومياً**: عدد المناديب الذين يعملون فعلياً
3. **متوسط ساعات العمل**: لمناديبه
4. **متوسط ساعات الاستراحة**: لمناديبه
5. **نسبة النشطين**: (النشطين / الإجمالي) × 100
6. **نسبة الغياب**: (الغائبين / الإجمالي) × 100

#### تصنيف الأداء

| التقييم | الشروط |
|---------|--------|
| ⭐ **ممتاز** | متوسط الساعات ≥ 6 **و** نسبة النشطين ≥ 85% |
| ✅ **جيد** | متوسط الساعات ≥ 5.5 **و** نسبة النشطين ≥ 80% |
| ⚠️ **متوسط** | متوسط الساعات ≥ 5 |
| 🔴 **ضعيف** | متوسط الساعات < 5 **أو** نسبة النشطين < 80% |

#### المخرجات
- **قائمة مرتبة** بجميع المشرفين وتقييمهم
- **بطاقة تفصيلية** لكل مشرف بجميع المؤشرات
- **توصيات**:
  - المشرفين الأقوياء (يمكن الاعتماد عليهم للتوسع)
  - المشرفين الضعفاء (يحتاجون تدريب أو استبدال)

### 6️⃣ **حساب Delta (نمو القوى العاملة)**

#### المدخلات
من Google Sheets:
- **تبويب "طلبات التعيين"**: التعيينات الجديدة
- **تبويب "طلبات إعادة التفعيل"**: المناديب المعاد تفعيلهم
- **تبويب "طلبات الإقالة"**: المناديب المقالين

#### المخرجات
```
التعيينات الجديدة: +45
إعادة التفعيل: +12
الإقالات: -23

Delta (التغير الصافي) = +45 + 12 - 23 = +34 مندوب
النمو الصافي = +34 مندوب
```

#### الفائدة
- معرفة النمو الفعلي في عدد المناديب
- مقارنة معدلات التوظيف بالفقد
- التخطيط للاحتياجات المستقبلية

## 🖥️ **واجهة المستخدم**

### الصفحة الرئيسية: `/admin/logistics-intelligence`

#### 1. **الفلاتر**
```
┌─────────────────────────────────────────────────────┐
│ من تاريخ  │ إلى تاريخ │ Target │  تشغيل التحليل   │
│ 2026-07-01 │ 2026-07-15 │  1500  │      📊 تشغيل      │
└─────────────────────────────────────────────────────┘
```

#### 2. **ملخص Delta**
```
┌─────────────────────────────────────────────────────┐
│ 📈 التغير الصافي في القوى العاملة (Delta)        │
│                                                     │
│  +34 مندوب                                         │
│                                                     │
│  تعيينات جديدة: +45  │  إعادة تفعيل: +12  │  إقالات: -23  │
└─────────────────────────────────────────────────────┘
```

#### 3. **المؤشرات الرئيسية**
```
┌──────────────┬──────────────┬──────────────┬──────────────┐
│ إجمالي       │ متوسط النشطين│ متوسط الساعات│ إجمالي الساعات│
│ المناديب     │   يومياً     │              │              │
│             │              │              │              │
│    250      │    200       │  5.9 ساعة    │  7080 ساعة   │
└──────────────┴──────────────┴──────────────┴──────────────┘
```

#### 4. **التبويبات (Tabs)**

**📊 نظرة عامة (Overview)**
- رسم بياني للاتجاه اليومي (Active, Absent, Hours)
- توزيع شرائح ساعات العمل (Pie Chart)

**📈 شرائح الأداء (Segments)**
- عرض مفصل لكل شريحة (< 4h, 4-6h, 6-8h, 8+h)
- قائمة المناديب في كل شريحة

**⚠️ المشاكل (Issues)**
- المناديب غير النشطين 3+ أيام
- Top 10 متخذين للاستراحات
- Top 10 غائبين
- زر "تحليل المشاكل وإنشاء تحذيرات"
- عرض تحذيرات الأداء مع الحلول

**🎯 التخطيط (Planning)**
- حاسبة Target
- مولد خطط النمو

**👔 المشرفين (Supervisors)**
- قائمة جميع المشرفين مع تقييمهم
- مؤشرات الأداء لكل مشرف

## 📈 **أمثلة الاستخدام العملي**

### سيناريو 1: تحليل الأداء الأسبوعي
```
1. افتح الصفحة /admin/logistics-intelligence
2. اختر الفترة: آخر 7 أيام
3. اضغط "تشغيل التحليل"
4. راجع المؤشرات الرئيسية
5. اذهب لتبويب "المشاكل"
6. اضغط "تحليل المشاكل وإنشاء تحذيرات"
7. راجع التحذيرات والحلول المقترحة
```

### سيناريو 2: تخطيط التوسع
```
1. شغل التحليل للشهر الحالي
2. اذهب لتبويب "التخطيط"
3. أدخل Target النهائي: 2000 ساعة
4. أدخل الإطار الزمني: 4 شهور
5. اضغط "إنشاء خطة النمو"
6. ستحصل على خطة شهرية مفصلة:
   - عدد المناديب المطلوبين كل شهر
   - عدد التعيينات الجديدة
   - الأهداف والمعالم
```

### سيناريو 3: تقييم مشرف جديد
```
1. شغل التحليل لآخر 30 يوم
2. اذهب لتبويب "المشرفين"
3. ابحث عن اسم المشرف
4. راجع المؤشرات:
   - نسبة النشطين
   - متوسط ساعات العمل
   - نسبة الغياب
5. قارنه بالمشرفين الآخرين
6. حدد إذا كان يحتاج تدريب أو دعم
```

## 👨‍💼 **للمدير التشغيلي (كمدير لوجستيك)**

هذا النظام صُمم خصيصاً ليكون **المساعد الذكي** لك. استخدمه لـ:

✅ **فهم الوضع الحالي بسرعة**
   - كم مندوب نشط؟
   - كم الإنتاجية؟
   - أين المشاكل؟

✅ **اتخاذ قرارات مبنية على بيانات**

✅ **التخطيط الاستراتيجي**
